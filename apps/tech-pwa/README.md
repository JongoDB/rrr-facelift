# @rrr/tech-pwa

Phone-installable PWA for technicians. Hosted at `app.triple-r-rv.com`.

Voice-first ergonomics: tap mic, speak the job, review extracted line items, push to Zoho Books as estimate or invoice.

## Phase 00 status

Workspace stub only. The full Vite + React + Tailwind + shadcn/ui project, magic-link auth, chat UI, and voice flow are built in **Phases 04 (chat) and 05 (voice)** per planning/08-tech-pwa-spec.md.

## Why so empty right now

Same reason as `apps/web`: don't pin Vite/React versions before they're used. Phase 04 starts with `pnpm create vite` in this directory.
