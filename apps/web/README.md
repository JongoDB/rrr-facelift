# @rrr/web

Marketing website hosted at `triple-r-rv.com`. Astro static site, deployed to Cloudflare Pages.

## Phase 00 status

Workspace stub only. The full Astro project, content collections, intake form (React island), Cloudflare Pages deploy config, and DNS migration plan are built in **Phase 03 — Website Migration** per planning/09-website-migration.md.

## Why so empty right now

Adding Astro/Tailwind/etc here without using them yet would pin versions before we need to. Phase 03 starts with a fresh `pnpm create astro` inside this directory, then wires up content from the Squarespace snapshot script.
